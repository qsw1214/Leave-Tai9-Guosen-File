# vuedemo

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
window: npm run ent-dev
mac: npm run ent-dev-mac

# build for production with minification
npm run build
window: npm run ent-build
mac: npm run ent-build-mac
```
